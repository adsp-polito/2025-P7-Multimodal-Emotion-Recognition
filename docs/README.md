# Documentation Folder

This directory contains all project documentation for the **Multimodal Emotion Recognition Using Large Language Models** project. It includes design materials, management artifacts, functional specifications, checkpoint presentations, and supporting diagrams. The structure reflects the four-pillar methodology used throughout the project: Design, Manage, Develop, and Communicate.

---

## Folder Structure

### 1. Project Overview  
High-level descriptions of the project including vision, objectives, value proposition, and scope.  
- `project_overview.md`

### 2. Design Documents  
Documentation related to system design and conceptual modeling, including stakeholder analysis, personas, functional requirements, and the functional diagram.    
- `stakeholders_map/`  
- `personas/`  
- `functional_diagram/`

### 3. Checkpoint Presentations  
All presentations delivered during the course. Each checkpoint folder contains the completed slide deck and visual assets.  
- `checkpoints/CP1_slides/`  
- `checkpoints/CP2_slides/`  
- `checkpoints/CP3_slides/`

### 4. Gantt and Planning Materials  
Project planning resources, such as the Gantt chart and related timelines.  
- `gantt/`

### 5. Additional Documentation  
Any supplemental files or references that support the project but do not fit into the categories above.
